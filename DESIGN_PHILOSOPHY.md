# 마이리얼트립 할인쿠폰 사이트 - 디자인 철학

## 선택된 디자인 철학: 신뢰성 기반의 현대적 정보 아키텍처

### 핵심 원칙

**1. 명확한 정보 계층 구조**
- 사용자가 한눈에 필요한 정보를 찾을 수 있도록 명확한 시각적 계층을 구성
- 중요도에 따라 크기, 색상, 위치를 전략적으로 배치
- 카드 기반 레이아웃으로 각 할인 정보를 독립적이면서도 연결된 형태로 표현

**2. 신뢰감과 전문성**
- 금융/여행 정보를 다루는 사이트이므로 안정적이고 신뢰할 수 있는 비주얼 필요
- 과도한 장식 제거, 깔끔한 타이포그래피로 전문성 강조
- 공식 로고와 카드사 로고를 적절히 배치하여 신뢰성 강화

**3. 효율적인 스캔 가능성**
- 사용자가 빠르게 정보를 스캔할 수 있도록 설계
- 테이블, 비교 차트, 강조 박스 등 다양한 정보 표현 방식 활용
- 색상 코딩으로 할인율, 조건 등을 직관적으로 구분

### 색상 철학

**Primary Color: 트래블 블루 (#1E40AF)**
- 마이리얼트립의 브랜드 색상 기반
- 신뢰감, 여행, 안정성을 상징
- CTA 버튼, 강조 텍스트에 사용

**Secondary Color: 골드 액센트 (#D97706)**
- 프리미엄감과 특별한 혜택을 표현
- 할인율, 특가 정보 강조
- 호버 상태 및 특별 섹션에 사용

**Neutral Palette: 그레이 스케일**
- 배경: #F9FAFB (밝은 회색)
- 텍스트: #111827 (깊은 검정)
- 보조 텍스트: #6B7280 (중간 회색)
- 보더: #E5E7EB (연한 회색)

### 레이아웃 패러다임

**비대칭 구조 기반 정보 아키텍처**
- 메인 콘텐츠 영역과 사이드 네비게이션의 비대칭 배치
- 각 섹션별로 다양한 레이아웃 변형 (카드 그리드, 테이블, 타임라인 등)
- 모바일에서는 스택 구조로 자연스럽게 변환

### 시그니처 요소

**1. 할인 배지 (Discount Badge)**
- 각 할인 정보의 금액을 강조하는 원형 또는 사각형 배지
- 색상으로 할인 규모 구분 (5,000원+: 골드, 3,000원: 실버, 1,000원: 브론즈)

**2. 카드 컴포넌트**
- 그림자와 호버 효과로 깊이감 표현
- 각 카드는 독립적인 정보 단위로 기능
- 카드 상단에 카드사/서비스 로고 배치

**3. 비교 테이블**
- 깔끔한 보더와 줄무늬 패턴으로 가독성 향상
- 조건 충족 시 초록색, 미충족 시 회색으로 시각화
- 모바일에서는 카드 형식으로 변환

### 상호작용 철학

**버튼 상태 설계**
- Default: 파란색 배경, 흰색 텍스트
- Hover: 진한 파란색, 미묘한 그림자 추가
- Active: 더 진한 파란색, 내부 그림자
- Disabled: 회색, 낮은 불투명도

**복사 기능 피드백**
- 할인 코드 복사 시 토스트 알림으로 즉시 피드백
- "복사됨!" 메시지와 함께 1.5초 표시 후 자동 사라짐

**앵커 링크 스크롤**
- 목차에서 섹션으로 이동 시 부드러운 스크롤 애니메이션
- 현재 섹션을 목차에서 하이라이트

### 애니메이션 가이드

**진입 애니메이션**
- 페이지 로드 시 섹션이 아래에서 위로 페이드인 (200ms)
- 카드는 약간의 지연(stagger)을 두고 순차적으로 나타남

**호버 애니메이션**
- 카드: 약간 위로 올라오는 효과 (transform: translateY(-4px), 150ms)
- 버튼: 배경색 변화 (150ms ease-in-out)
- 링크: 밑줄 애니메이션 (width 변화)

**로딩 상태**
- 스켈레톤 로더 사용 (실제 콘텐츠 크기와 동일한 형태)
- 부드러운 펄싱 애니메이션

### 타이포그래피 시스템

**Display Font: Pretendard Bold**
- 페이지 제목, 섹션 헤더에 사용
- 크기: 32px-48px
- 라인 높이: 1.2

**Heading Font: Pretendard SemiBold**
- 서브 헤더, 카드 제목에 사용
- 크기: 18px-24px
- 라인 높이: 1.4

**Body Font: Pretendard Regular**
- 본문 텍스트
- 크기: 14px-16px
- 라인 높이: 1.6

**Accent Font: Pretendard Bold**
- 강조 텍스트, 배지
- 크기: 12px-14px
- 라인 높이: 1.5

### 접근성 고려사항

- 모든 텍스트는 최소 4.5:1 명도 대비 비율 준수
- 색상만으로 정보 전달하지 않음 (아이콘, 텍스트 라벨 병행)
- 포커스 상태 명확하게 표시
- 모바일 터치 타겟 최소 44x44px 확보

### 반응형 디자인

**Breakpoints**
- Mobile: 320px-639px
- Tablet: 640px-1023px
- Desktop: 1024px+

**모바일 최적화**
- 카드는 1열 또는 2열로 표시
- 테이블은 카드 형식으로 변환
- 네비게이션은 햄버거 메뉴로 축소
- 폰트 크기 조정으로 가독성 유지
